Predicting Time-to-Failure of Water Mains in Water Distribution Network in Scotland – A Machine Learning Approach

This is a master's project that aims to predict time to next failure (burst) of water mains across Scotland. It is done in collaboration with Scottish Water and Edinburgh Napier University. The project will use different machine learning models.

The data contains attributes such as pipe diamater, length, age, soil type, surface code among others.
